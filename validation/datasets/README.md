# Validation Datasets

## Cohen 2006 Benchmark
- **Source**: Cohen et al. "Reducing Workload in Systematic Review Preparation Using Automated Citation Classification" (JAMIA, 2006)
- **Content**: ~14,000 papers across 15 systematic review topics
- **Download**: `python validation/datasets/download_cohen.py`

## ASReview Synergy
- **Source**: ASReview Synergy Dataset (https://github.com/asreview/synergy-dataset)
- **Content**: 26+ labeled systematic review datasets
- **Download**: `python validation/datasets/download_asreview.py`
