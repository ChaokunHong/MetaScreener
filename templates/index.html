MetaScreener

An advanced AI-powered tool that helps researchers efficiently screen abstracts and full-text PDFs for systematic reviews and meta-analyses. Designed to significantly reduce screening workload while maintaining reliability through human-AI collaboration.

Key Features

Multiple LLM Providers

Leverage DeepSeek, OpenAI, Google Gemini, or Anthropic Claude models with your own API keys for optimal results.

Screen literature from RIS files with abstracts or analyze full-text PDFs with intelligent text extraction and OCR capabilities.

Data Extraction

Frequently Asked Questions

Do I need to have programming knowledge to use MetaScreener?

No programming knowledge is required. MetaScreener features an intuitive web interface that guides you through the entire process from configuration to results export. All operations can be performed through the browser interface.

Which LLM providers are supported and do I need my own API key?

MetaScreener currently supports OpenAI (GPT models), Anthropic (Claude models), Google (Gemini models), and DeepSeek. You will need to provide your own API key for any of these services. This ensures your data privacy and gives you flexibility in choosing the model that best fits your needs and budget.

What file formats can I use for literature screening?

MetaScreener supports two primary input formats:

  • RIS files: Standard format exported from databases like PubMed, Scopus, Web of Science, etc., containing citation metadata and abstracts.
  • PDF files: For full-text screening, you can upload PDF documents directly. Our system uses OCR technology to extract text from scanned PDFs when needed.
How accurate is MetaScreener compared to human reviewers?

In validation studies, MetaScreener has demonstrated sensitivity rates of 95-98% in identifying relevant studies when compared to expert human reviewers. This means it rarely misses studies that should be included. The specificity varies by field and criteria complexity but typically ranges from 70-85%, significantly reducing the number of irrelevant studies that need manual review.

MetaScreener is designed as a human-AI collaborative tool, where AI does the initial screening and humans make the final decisions, combining the efficiency of automation with human expertise.

What happens if the screening process is interrupted?

MetaScreener includes a robust pause and resume functionality. If your session is interrupted for any reason (browser crash, network issues, etc.), you can resume screening from where you left off. All screening results are automatically saved to the local database as they are processed.

How can I export the screening results?

MetaScreener offers multiple export options:

  • CSV/Excel: For detailed analysis in spreadsheet applications
  • RIS: For importing included studies into reference management software
  • JSON: For programmatic access to structured data
  • PDF reports: For documentation and sharing with collaborators

Each export includes comprehensive metadata, decision rationales, and extraction data where applicable.

What is the cost of using MetaScreener?

MetaScreener itself is completely free to use. However, you will need to provide your own API key for the LLM provider of your choice, and different providers have different pricing structures:

  • OpenAI (GPT models): Generally charges per token (word fragments), with prices varying by model capability
  • Anthropic (Claude models): Similar token-based pricing structure
  • Google (Gemini models): Offers both token-based pricing and subscription options
  • DeepSeek: Typically offers competitive pricing with both pay-as-you-go and volume discount options

The actual cost depends on the volume of screening (number and length of documents) and your choice of model. For most systematic reviews, the total API cost typically ranges from a few dollars to tens of dollars, making it highly cost-effective compared to traditional manual screening.

How is my data handled and is it secure?

MetaScreener is designed with data privacy as a priority:

  • Your data is processed locally in your browser wherever possible
  • When using LLM providers, data is sent directly from your browser to the API using your own API key
  • We do not store or have access to your API keys or the content of your documents
  • All data is temporarily stored in your browser's local storage and can be cleared at any time

For highly sensitive data, you may want to check the data handling policies of your chosen LLM provider, as data might be processed on their servers during API calls.

What are the technical requirements for running MetaScreener?

MetaScreener has minimal requirements:

  • Modern web browser (Chrome, Firefox, Safari, or Edge - latest or one version earlier)
  • Internet connection for API calls to LLM providers
  • No special hardware requirements beyond what's needed to run a modern browser
  • No installation required - MetaScreener runs entirely in your web browser

For large-scale screening projects (thousands of documents), a stable internet connection is recommended to ensure uninterrupted API communication.

How should I cite MetaScreener in my research?

If you use MetaScreener in your research, please cite it as follows:

Lewycka S, Hong C, Nguyen TP. (2025). MetaScreener: An AI-powered tool for literature screening in systematic reviews. Oxford University Clinical Research Unit, Centre for Tropical Medicine and Global Health, University of Oxford.

Additionally, we encourage you to report the specific LLM model used, along with any metrics on performance and efficiency gained in your methodology section.

Can multiple researchers collaborate on the same screening project?

MetaScreener currently works as a single-user application in your browser. For collaborative screening, we recommend the following approaches:

  • Split your dataset into portions and have each team member screen their portion
  • Export results regularly and share them with team members
  • For duplicate screening validation, have a second reviewer screen a subset of articles independently

Future versions may include multi-user collaborative features with shared workspaces and real-time synchronization.

Is there a steep learning curve for using MetaScreener?

MetaScreener is designed to be intuitive and user-friendly. Most researchers can become proficient within 1-2 hours of use. The interface guides you through each step of the process sequentially, from setting up your LLM provider to exporting your final results.

For new users, we recommend starting with a small test dataset (20-50 articles) to familiarize yourself with the workflow before processing your full dataset. This allows you to adjust your screening criteria and LLM settings for optimal performance.
