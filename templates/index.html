Key Features

Multiple LLM Providers

Leverage DeepSeek, OpenAI, Google Gemini, or Anthropic Claude models with your own API keys for optimal results.

Flexible Screening Frameworks

Choose from PICOT, PICOS, SPIDER, PECO and other evidence-based frameworks to structure your review criteria.

Multi-Format Support

Screen literature from RIS files with abstracts or analyze full-text PDFs with intelligent text extraction and OCR capabilities.

Accelerated Workflow

Process multiple documents simultaneously with multi-threading and real-time progress tracking.

Performance Metrics

Evaluate screening accuracy with confusion matrices, Cohen's Kappa, sensitivity, specificity, and workload reduction statistics.

Data Extraction

Extract structured data from full-text documents with customizable extraction templates and JSON formatting.
